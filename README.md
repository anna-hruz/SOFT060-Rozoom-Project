# Rozoom — Flashcard Desktop Application

A desktop flashcard application built in Python as part of the SOFT060 
Software Development and Data Science Project module at the 
University of Plymouth (Academic Year 2025/2026).



## What is Rozoom?

Rozoom is a locally run flashcard quiz tool designed to help students 
memorise technical terminology through active recall. It was built 
with cybersecurity students in mind but works for any subject area.

There is no internet connection required, no account needed, and no 
data ever leaves your machine.



## Features

- Choose a category to study (e.g. AI, Cybersecurity)
- Quiz mode — type your answer and get instant correct/wrong feedback
- Score tracking — see your final score at the end of each quiz
- Shuffled questions — different order every time
- Add your own flashcards through an in-app form
- Edit or delete existing flashcards
- Scrollable manage screen for large card sets
- Data saved locally in a JSON file between sessions



## How to Run

**Requirements:** Python 3 must be installed on your machine.

**Mac users** — tkinter may need to be installed separately:
brew install python-tk@3.13
